# A Plugin for JKook
## PluginsManager



If you'd like to code your own JKook Plugin
Add this to your build.gradle.kts

```java
repositories {
    mavenCentral()
       maven {
           url = uri("https://www.xn--lzs769c.icu/repository/repository/")
       }


}

dependencies {
    implementation("com.github.SNWCreations:JKook:0.52.1")
}
```
